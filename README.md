PolyVest AI - Funding Analyst :

PolyVest AI is an AI-powered funding companion that helps startups analyze funding schemes, generate structured reports, and provide funding readiness insights based on uploaded documents or official funding databases.
Built with Streamlit, LlamaIndex, Groq LLM, and HuggingFace embeddings, it generates downloadable PDF reports and PPT slides from your queries.

Ask about startup funding schemes and eligibility

 Upload pitch decks or reports for AI analysis
 
 Generate structured Funding Readiness Reports:
   Eligibility check
   Valuation sanity check
   Investor match
   Red flags

 Download analysis as PDF or PPTX
Session-based chat interface with memory of conversation
